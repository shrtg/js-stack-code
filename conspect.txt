---- 02
при установке eslint надо сначала ставить нужный конфиг, и смотреть на странице в гитхабе что и как ставить

eslint-plugin-compat - проверяет доступны ли те или иные фишки в браузерах, указанных в package.json

Flow
// yarn add --dev flow-bin babel-preset-flow babel-eslint eslint-plugin-flowtype
сам Flow и плагины для babel и eslint
в конфиге eslint нужно указать
  // "extends": [..., "plugin:flowtype/recommended"]
что равнозначно
  // "parser": "babel-eslint"
т.е указывает eslint использовать парсер babel'я
.flowconfig
  // [options]
  // suppress_comment= \\(.\\|\n\\)*\\flow-disable-next-line
  задает комментарий для игнора предупреждений в следующей строке

// @flow
комеентарий который указывает что в файле надо проверять типы

Jest
// add --dev jest babel-jest

Husky
приблуда для гит-хуков

---- 03
папки src/client, src/server, src/shared -
  соотвтственно для клиентского, серверного и изоморфного(универсального) кода

compression - middleware для express для gzip сжатия на сервере

nodemon - утилита для автоматического перезапуска сервера Node при изменении файлов

PM2 - менеджер процессов для Node, обеспечивающий жизнеспособность вашего приложения и предлагающий тонны возможностей по управлению и мониторингу.

cross-env - кроссплатформенная установка переменных окружения

---- 04
babel-polyfill - использование ES6 полифиллов (Promise, etc)

Webpack
webpack.config.babel.js
  .babel.js - сообщает webpack что файл надо пропускать через babel-трансформации
babel-loader - babel-плагин для webpack

webpack-dev-server - запускает дев-сервер, (отслеживает изменения в клиентских файлах)
  надо держать открытым на другом порту, на отдельной вкладке терминала

---- 05

ImmutableJS (или просто Immutable) библиотека разработанная компанией Facebook для манипулирования неизменяемыми коллекциями,
например списки и итерируемые объекты. Любое изменение неизменяемого объекта возвращает новый объект без изменения первоначального объекта.

isomorphic-fetch - кроссбраузерный (+ node) полифилл для fetch

redux-thunk - middleware for redux, для асинхронных действий
